# Expo Starter App

Expo SDK 55 canary with React 19, React Native 0.83, and expo-router.

## Get started

```bash
npm install
npm run ios      # or: npm run android
```

This project uses **canary SDK 55** which requires a [development build](https://docs.expo.dev/develop/development-builds/introduction/). Expo Go is not supported.
